Introduction

This is the latest documentation of XMLReader. If you want to know what XMLReader is check out the XMLReader Official Website. Terms used in class and function naming are based on XML so if you're unfamiliar with XML you might want to read a few tutorials.

Note: Below you will find documentation of all the public members of XMLReader classes and all independent functions defined in the XMLReader namespace. For each entry there is a definition, description and an example of how it can be used.

Documentation

  • Namespace

    • All functions and classes used in XMLReader are defined in the XMLReader namespace. This means that in order to access an element of XMLReader you'll need to use the following syntax: XMLReader::ElementName.

  • Classes

    • Attribute

      • Description:

        • XMLReader::Attribute is a class that resembles an attribute of an XML tag. An XMLReader::Attribute is fully described by two variables, Name and Value.

      • Public Methods:

            • XMLReader::Attribute::Attribute()
            XMLReader::Attribute::Attribute(string Name, string Value);

          • Description:

          • Use:

            • XMLReader::Attribute Attribute ("Name", "Value");
            //Or just XMLReader::Attribute Attribute;

        • ~Attribute()

          • Definition:

            • XMLReader::Attribute::~Attribute()

          • Description:

            • Destructor. Used to clean up after the object is no longer necessary.

          • Use:

          • Definition:

            • void XMLReader::Attribute::Create(string &Name, int &Trigger)

          • Description:

      • Description:

        • XMLReader::Tag is a class that resembles a tag of an XML structure. An XMLReader::Tag has a vector of XMLReader::Attributes that can be browsed through with public methods. An XMLReader::Tag is fully described by three variables, Name, Trigger and Text.

      • Public Methods:

        • Tag()

          • Definition:

            • XMLReader::Tag::Tag()
            XMLReader::Tag::Tag(string Name, int Trigger)

          • Description:

            • Constructors. Two definitions, second one also specifies the tag's name and trigger.

          • Use:

            • XMLReader::Tag Tag ("Name", 2);
            //Or just XMLReader::Tag Tag;

        • ~Tag()

          • Definition:

            • XMLReader::Tag::~Tag()

          • Description:

            • Destructor. Used to clean up after the object is no longer necessary.

          • Use:

            //Do stuff

        • Create()

            • void XMLReader::Tag::Create(string &Name, string &Value)

          • Use:

          • Definition:

            • string &XMLReader::Tag::GetName()

            Note: You should use a constant reference to avoid accidentally changing the tag's properties.

        • GetTrigger()

          • Definition:

            • int &XMLReader::Attribute::GetValue()

          • Description:

            • Returns a reference to the tag's Trigger.

          • Use:

            • //Already initialized XMLReader::Tag::Tag
            const int &Trigger = Tag.GetTrigger();

            Note: You should use a constant reference to avoid accidentally changing the tag's properties.
            Note 2: XMLReader::Tag::Trigger defines the nature of the tag. It is an integer and has the following possible values: 0, if the tag is a close tag (</Tag>), 1, if it's an open tag (<Tag>), 2, if the tag is self-closing (<Tag />), 3 if the tag is a declaration tag (<?xml ?>).

        • GetText()

          • Definition:

            • string &XMLReader::Tag::GetText()

          • Description:

            • Returns a reference to the tag's text.

          • Use:

            • //Already initialized XMLReader::Tag Tag
            const string &Text = Tag.GetText();

            Note: You should use a constant reference to avoid accidentally changing the tag's properties.
            Note 2: XMLReader::Tag::Text contains any non-tag string between an opening and a closing tag. Example: <Tag>Text</Tag>.

    • File

      • Description:

        • XMLReader::File is a class that resembles an XML structure. An XMLReader::File has a vector of XMLReader::Tags that can be browsed through with public methods.

      • Public Methods:

          • Definition:

            • XMLReader::File::File()
            XMLReader::File(string Location, string Root = "")

          • Description:

          • Use:

            • XMLReader::File File ("MyData.xml");
            //Or just XMLReader::File File;

        • ~File()

          • Definition:

            • XMLReader::File::~File()

          • Description:

          • Use:

          • Use:

            • XMLReader::File File;

            if (File.Parse(Location)) {
                //Do stuff
            }

  • Functions

    • FindMinimum()

      • Definition:

        • int XMLReader::FindMinimum(int Integers[], int Size)

      • Description:

        • Returns the minimum value from an array of integers. The size of the array is passed as an argument.

      • Use:

        • int Integers[4] = {0, 1, 42, 6124};
        int Minimum = XMLReader::FindMinimum(Integers, 4);

    • FindTotalOf()

      • Definition:

        • int XMLReader::FindTotalOf(char Character, const string &String)

      • Description:

        • Returns the total occurences of Character in String.

      • Use:

        • string String = "1;2;3;4;5;6;7";
        int TotalSemiColons = XMLReader::FindTotalOf(';', String);

    • FindNthOf()

      • Definition:

        • int XMLReader::FindNthOf(int &Occurence, char Character, const string &String)

      • Description:

        • Returns position of the nth occurence of Character in String, where n = Occurence. The position of the (n+1)th character is also returned in Occurence. Returns -1 if not found.

      • Use:

        • string String = "1;2;3;4;5;6;7";
        int TotalSemiColons = XMLReader::FindTotalOf(';', String);
        int i = 3; int ThirdSemiColon = XMLReader::FindNthOf(i, ';', String);
        //i now holds the position of the fourth semicolon or -1 if there is no next occurence.

    • String2()

      • Definition:

        • template <class Type>
        Type XMLReader::String2(const string &String, char Seperator = '\0', int SubstringNumber = 0);

      • Description:

        • Converts a string to Type and returns the converted value. The seperator argument is used for strings like '800x600', in which the character 'x' is the seperator. If you use a seperator you must also specify which substring you want to return, 0 being 800 and 1 being 600, in the previous example. By default the first substring is returned.

      • Use:

        • //Without seperator
        string String = "2011";
        int Integer = XMLReader::String2<int>(String);

        //With seperator
        string OtherString = "0;1;2;3;4;5;6";
        int OtherInteger = XMLReader::String2<int>(OtherString, ';', 2); //this will the value between the second and third semicolons, 2.

        Note: Built-in valid types are int, bool, char, float.
